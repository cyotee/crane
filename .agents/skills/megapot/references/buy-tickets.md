# Buy tickets with custom numbers

**Not supported in this skill.** Purchasing tickets with specific/custom numbers is only available at https://megapot.io.

If the user asks to pick their own numbers (e.g. "buy a ticket with numbers 1 6 16 22 25 bonus 4", "lemme pick my own numbers"), tell them:

> Choosing specific ticket numbers is currently only available on megapot.io. I can buy you quick-pick (random) tickets instead — would you like me to do that?

For random tickets, see `references/buy-random.md`.
